Элемент сообщения в чате <Message />:

Внешний вид bubble, слева размещен аватар автора сообщения (аватар отображается только в групповом чате). У своих сообщений аватар не отображается.

Если переписка ведётся только с двумя участниками, то аватар не отображается.

В своих сообщениях аватар не отображается.

Имя автора сообщения отображается сверху bubble (если это группа сообщений, то только в начале группы)

Нижний угол стороны автора не должен закругляться. Если отправлено несколько сообщений подряд от одного автора, то углы не закругляем.

1 вид: Закруглена верхняя часть края.

2 вид: Нет закруглений с краю.

Время сообщения отображается справа/слева от сообщения при наведении на сообщение.

При пролистывании блока из нескольких сообщений или длинного сообщения, аватар автора сообщений приклеивается к низу до конца блока с сообщениями.

Сообщение имеет индикацию состояния.

ПКМ на сообщении - открывает список возможных действий с сообщением (Добавить реакцию, ответить, переслать, удалить, изменить).

Сообщение может пересланное или с ответом.
